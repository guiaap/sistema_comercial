# Sistema Comercial

Sistema web full-stack desenvolvido para a disciplina de Programação Web II do curso de Sistemas de Informação da Uniavan.

O projeto tem como objetivo simular um sistema comercial real, permitindo o gerenciamento de informações empresariais através de uma aplicação web construída com Java e Spring Boot.

## Autor

Guilherme Amaral Azevedo Pedroso

## Tecnologias Utilizadas

### Frontend

- HTML5
- CSS3
- JavaScript
- Bootstrap 5

### Backend

- Java 21
- Spring Boot
- Spring Web
- Spring Data JPA
- Maven
- PostgreSQL

## Funcionalidades

- Cadastro de clientes
- Cadastro de usuários
- Controle de produtos
- Gerenciamento de vendas
- Controle de estoque

## Como Executar

### Pré-requisitos

- Java 21
- Maven
- PostgreSQL

### Execução

```bash
git clone <url-do-repositorio>
cd sistema-comercial
mvn spring-boot:run
